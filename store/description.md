# Chrome Web Store 掲載情報

## 拡張機能名
おとり物件バスター

## 短い説明（132文字以内）
不動産サイトの物件をリアルタイム分析し、おとり物件の可能性をスコア表示。SUUMO・HOME'S・at home・CHINTAI・Yahoo!不動産に対応。

## 詳細説明

おとり物件バスター は、賃貸物件サイトを閲覧中に「おとり物件」の可能性をリアルタイムで判定・表示するChrome拡張機能です。

### おとり物件とは？
実際には契約できない物件を広告に掲載し、来店した客に別の物件を紹介する違法な営業手法です。宅建業法で禁止されていますが、いまだに多くのサイトに掲載されています。

### 主な機能

**スコア表示**
各物件カードにおとりスコア（0〜100点）をバッジ表示。一目で危険度がわかります。

**6つの判定因子**
1. 相場との乖離 - 地域・間取り別の相場データと比較
2. 住所の曖昧さ - 番地が記載されているか
3. 物件名の有無 - 具体的な物件名があるか
4. 写真の枚数 - 十分な写真が掲載されているか
5. 極端な低価格 - 常識的にありえない家賃ではないか
6. 他ユーザーの通報件数 - 他のユーザーからの報告数

**通報機能**
おとり物件を発見したら、ワンクリックで通報できます。通報データは集計され、他のユーザーのスコアリングに反映されます。

**公式通報リンク**
国土交通省、不動産公正取引協議会、消費者ホットラインなど、おとり物件を公式に通報できる窓口へのリンクを提供しています。

### 対応サイト
- SUUMO（スーモ）
- LIFULL HOME'S（ホームズ）
- at home（アットホーム）
- CHINTAI（チンタイ）
- Yahoo!不動産

### プライバシー
- 物件データはブラウザ内で処理され、自動的に外部送信されることはありません
- 通報はユーザーの明示的な操作（ボタンクリック）がある場合のみ送信されます
- 個人を特定する情報は収集しません
- 詳しくはプライバシーポリシーをご確認ください

### アフィリエイト広告について
本拡張機能にはAmazonアソシエイト・プログラムのアフィリエイトリンクが含まれています。ポップアップおよびスコア詳細パネルに「広告 - Amazonアソシエイト」と明記されたリンクが表示されます。これらのリンクをクリックしてAmazonで商品を購入された場合、開発者が紹介料を受け取ることがあります。アフィリエイトリンクの利用は完全に任意であり、拡張機能の基本機能（おとり物件スコアリング・通報）には一切影響しません。

### 免責事項
本拡張機能が表示するスコアは参考値であり、物件がおとりであることを断定するものではありません。最終的な判断はご自身の責任で行ってください。

## host_permissions の理由説明（審査用）
本拡張機能は、ユーザーが「この物件を通報する」ボタンをクリックした際にのみ、通報データを自社サーバーに送信します。また、他のユーザーによる通報件数を取得し、おとり物件のスコアリングに反映するために使用します。自動的なデータ収集は一切行いません。

## content_scripts の理由説明（審査用）
対応する不動産ポータルサイト（SUUMO、HOME'S、at home、CHINTAI、Yahoo!不動産）の物件一覧ページにおいて、各物件の情報（家賃、住所、写真枚数等）を解析し、おとり物件の可能性をスコア表示するために使用します。解析はブラウザ内で完結し、ページの内容を改変することはありません（自拡張のUIをオーバーレイ表示するのみです）。
